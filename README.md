# Blog Cá Nhân

Đây là mã nguồn cho blog cá nhân được xây dựng bằng Astro và Tailwind CSS.

## Cài đặt

```bash
npm install
npm run dev
```

## Tính năng

- Thiết kế responsive với Tailwind CSS
- Hiệu ứng Glass Morphism
- Hệ thống bình luận
- Tìm kiếm bài viết
- Giao diện quản trị
- Trình soạn thảo Markdown
- Quản lý bài viết và bình luận
